隐私政策

欢迎使用 Mola相机。本应用由 上海视界象限科技有限公司（以下简称"我们"）开发与运营。我们重视您的隐私与安全。本政策说明我们如何收集、使用、存储与共享与您相关的信息。使用本应用即表示您理解并同意本政策；若您不同意，请停止使用本应用。

【特别提示：我们处理的个人敏感信息】

为实现拍摄、登录、支付与故障排查等功能，本应用及所接入的第三方 SDK 会处理以下个人敏感信息，特向您突出明示：

（1）设备标识信息：设备 MAC 地址、设备序列号、IMEI、Android ID、OAID 等设备识别码；

（2）手机号码：您在登录时填写的本人手机号，用于创建并登录账号、接收短信验证码；

（3）照片与图像信息：您拍摄或选择用于处理、保存、AI 分析的照片，及其中可能包含的人物图像。

上述敏感信息的收集目的、收集方式与对应的第三方 SDK 明细，详见本政策第二、四、五节。除实现相应功能所必需外，我们不会收集与功能无关的敏感信息，也不会将其用于用户画像或广告推送。

一、当前账号与社区功能状态

本应用提供手机号 + 短信验证码登录，用于开通会员、记录 AI 使用额度等账号相关功能。登录时您需要填写本人手机号并接收短信验证码；未注册的手机号在首次验证通过后将自动创建账号（以您的手机号作为账号标识）。除登录所需外，本应用不提供广场/社区发布、评论、关注或公开个人主页功能，也不会在应用内运营向其他用户公开展示您照片或文字的社区功能。您可随时在「我的」页面注销账号，详见下文「九、您的权利」。

二、我们收集哪些信息

1. 您在使用中产生的内容

  • 照片：您通过相机拍摄或选择用于处理或保存的内容。
  • 文本：您输入的拍摄意图、AI 提示词及其他您主动输入的文字。

2. 账号与登录信息

  • 为提供登录与会员服务，我们会收集您登录时填写的手机号码，并向该号码发送短信验证码用于验证。手机号将作为您的账号标识，用于记录会员状态与 AI 使用额度。
  • 短信验证码由我们的服务器通过第三方短信服务（阿里云短信）代为下发，详见下文「五、第三方 SDK 目录」。我们不会读取您设备通讯录中的其他号码，也不会将您的手机号用于与登录、会员无关的营销或推送。

3. 设备与日志信息

  • 为保障服务质量、账号安全、支付风控与故障排查，我们及所接入的第三方 SDK 可能收集以下设备相关信息：设备标识信息（如设备 MAC 地址、设备序列号、IMEI、Android ID、OAID 等设备识别码）、设备型号、操作系统版本、设备设置、崩溃与运行日志等。
  • 为完成微信/支付宝分享与支付功能，相关支付与开放平台 SDK 可能检测您设备中是否已安装对应应用（读取本应用相关的已安装应用信息），用于判断能否唤起微信/支付宝完成支付或分享。
  • 上述信息不包含您的通讯录、短信内容或通话记录。设备标识与已安装应用检测信息主要由第三方 SDK 在其功能范围内收集与使用，详见下文「第三方 SDK 目录」。

4. 传感器信息

  • 为实现取景时的「水平仪」辅助线（提示画面是否水平），我们会读取设备加速度传感器（重力/加速度）数据，实时计算设备的倾斜（滚转）角度。
  • 收集方式：通过系统传感器接口实时读取，仅在您使用相机取景界面时读取，退出即停止。
  • 使用范围：该数据仅在您的设备本地内存中用于绘制水平参考线，不会被存储、不会上传至我们的服务器或任何第三方，也不用于识别您的身份或任何其他用途。

5. 应用内购买

  • 若开通会员等应用内购买，支付将通过应用内支付渠道（如支付宝、微信支付）完成；我们不会接收或存储您的完整银行卡号。为完成支付与风控，支付渠道 SDK 可能收集设备标识、MAC 地址、设备序列号及已安装应用信息，详见下文「第三方 SDK 目录」。

三、人脸数据（Face Data）

本应用不收集、不上传、不存储可用于识别您个人的生物特征数据（如人脸模板、深度图、面部几何网格等）。

为实现仅在设备端完成的取景引导与人像相关功能，我们可能使用系统提供的设备端人脸检测能力，在内存中获得人脸在画面中的大致位置（边界框），用于定位局部处理范围。该过程在您的手机上完成，人脸检测结果不会传输至我们的服务器。

人脸相关数据不会用于身份识别、用户画像、广告定向或与拍摄辅助无关的任何目的。设备端人脸检测产生的中间数据仅驻留在内存中，在处理完成后即被释放，不做持久化存储。

四、第三方 AI 服务（图像传输与用途）

当您主动使用 AI 拍照分析、AI 辅助或风格引导等功能时，我们会将该次功能涉及的图像通过 HTTPS 加密传输至第三方 AI 服务，用于生成文字建议或分析结果。

  • 服务提供方：当前为国内大模型服务商（包括但不限于阶跃星辰、阿里云通义千问 DashScope、火山引擎豆包）。
  • 传输内容：主要为您拍摄或选择用于分析的静态照片以及相关提示文本。我们不会传输您的通讯录、短信内容或与该功能无关的文件。
  • 本应用的目的：获取拍摄引导、构图与光线建议，并在应用内展示给您。
  • 您的控制：不使用 AI 功能时，相关云端分析不会执行；您仍可使用不依赖该服务的本地拍摄与基础功能。

服务方对传输数据的具体处理方式可能随其官方规则更新而变化，请查阅其最新公示文件。
